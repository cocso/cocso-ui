<Section>
  <Section.Text>
    Checkbox는 사용자가 여러 항목 중 하나 이상을 선택하거나 해제할 때 사용합니다. 선택, 해제, 중간 상태의 세 가지 상태를 지원합니다.
  </Section.Text>
</Section>

  </Section.Text>
  <ComponentExample name="checkbox-size" />
</Section>

<Section>
  <Section.Title>상태</Section.Title>

  </Section.Text>
  <ComponentExample name="checkbox-status" />
