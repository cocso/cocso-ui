<Section>
  <Section.Title>사용 방법</Section.Title>
  <Section.Text>
    `@cocso-ui/react-icons` 패키지에서 필요한 아이콘을 import하여 사용합니다.
  </Section.Text>

```tsx
import { SearchIcon, AddIcon } from '@cocso-ui/react-icons';

<SearchIcon size={20} />
<AddIcon size={24} />
```
