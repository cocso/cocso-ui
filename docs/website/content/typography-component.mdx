<Section>
  <Section.Text>Typography는 텍스트의 크기, 굵기, 행간 등을 일관되게 관리할 때 사용합니다. 반응형 폰트 크기를 지원하여 디바이스별로 최적화된 텍스트를 렌더링합니다.</Section.Text>
</Section>

  <Section.Title>Size</Section.Title>

</Section>

<Section>
  <Section.Title>Weight</Section.Title>
  <Section.Text>thin부터 black까지 아홉 가지 굵기를 지원합니다.</Section.Text>
  <ComponentExample name="typography-weight" />
</Section>
