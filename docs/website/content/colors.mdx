description: 브랜드 아이덴티티와 시각적 계층을 위한 색상 가이드
---

<Section>
  <Section.Text>
    색상은 브랜드를 인식시키고, 정보의 우선순위를 전달하며, 사용자에게 상태를 알리는 핵심 수단입니다. COCSO-UI는 일관된 색상 체계를 통해 모든 서비스에서 동일한 시각적 경험을 제공합니다.
  </Section.Text>
</Section>

<Section>
  <Section.Title>Color Palette</Section.Title>
  <Section.Text>
    5가지 색상 팔레트가 정의되어 있으며, 각 팔레트는 50부터 950까지 11단계의 명도를 포함합니다. Neutral은 텍스트와 배경에, Primary는 주요 액션에, Danger·Warning·Success는 상태 표시에 사용됩니다.
  </Section.Text>
  <ComponentExample name="color-palette" />
</Section>

<Section>
  <Section.Title>Semantic Colors</Section.Title>
  <Section.Text>
    시맨틱 컬러는 용도에 따라 팔레트에서 선택된 색상입니다. 텍스트 색상은 정보의 중요도에 따라 Primary, Secondary, Tertiary로 구분됩니다.
  </Section.Text>
  <ComponentExample name="color-semantic" />
</Section>

<Section>
  <Section.Title>사용 방법</Section.Title>
  <Section.Text>
    모든 색상은 CSS 변수로 제공됩니다. 컴포넌트 내부에서는 자동으로 적용되며, 커스텀 스타일링이 필요한 경우 `var(--ds-color-primary-500)` 형식으로 직접 참조할 수 있습니다.
  </Section.Text>
</Section>
