## `@cocso-ui`

`@cocso-ui`는 COCSO의 사용자 인터페이스(UI) 컴포넌트 라이브러리에요.

이 라이브러리는 COCSO의 디자인 시스템을 기반으로 하여 일관된 사용자 경험을 제공하는 것을 목표로 하고 있어요.
