## `@cocso-ui/react`

이 Package는 리액트 기반 컴포넌트를 제공해요.

`radix-ui`의 헤드리스 컴포넌트를 사용하여 접근성과 커스터마이징을 고려한 컴포넌트를 구현하고 있어요.

### Installation

```bash
pnpm install @cocso-ui/react
```

### Usage

컴포넌트를 아래와 같은 형태로 import 할 수 있어요.

```javascript
import {Button} from '@cocso-ui/react';
```
