## `@cocso-ui/react-native-icons`

이 Package는 `react-native-svg` 기반 React Native 아이콘 컴포넌트를 제공합니다.

### Installation

```bash
pnpm install @cocso-ui/react-native-icons
```

### Peer Dependencies

```bash
pnpm install react-native-svg
```

### Usage

컴포넌트를 아래와 같은 형태로 import 할 수 있습니다.

```jsx
import { SearchIcon } from '@cocso-ui/react-native-icons';
```
