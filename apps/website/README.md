## `@cocso-ui/website`

이 Package는 cocso 디자인 시스템의 공개 문서 사이트를 제공해요.

Next.js 16과 fumadocs 기반으로 컴포넌트 가이드, 디자인 토큰 레퍼런스, 시작 가이드를 제공해요.

### Usage

```bash
# 개발 서버 실행 (포트 6005)
pnpm --filter @cocso-ui/website dev

# 프로덕션 빌드
pnpm --filter @cocso-ui/website build
```
