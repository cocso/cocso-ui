<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>Spinner renders an accessible animated loading indicator. It announces a label to screen readers via `aria-label` and supports semantic color variants to match the surrounding context.</Section.Text>
</Section>

<Section>
  <Section.Title>Size</Section.Title>
  <Section.Text>Three sizes are available: `large`, `medium`, and `small`. The default is `medium`.</Section.Text>
  <ComponentExample name="spinner-default" />
</Section>

<Section>
  <Section.Title>Variant</Section.Title>
  <Section.Text>Seven color variants map to semantic meanings. Use `white` when displaying the spinner on a dark background.</Section.Text>
  <ComponentExample name="spinner-variants" />
</Section>
