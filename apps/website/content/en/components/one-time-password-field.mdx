---
title: OTP Field
description: A split input field for entering one-time passwords and verification codes.
---

<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>OneTimePasswordField renders a row of individual input slots for entering a one-time password or verification code. Built on input-otp, it supports paste, keyboard navigation, and accessibility out of the box. Use onValueChange to receive the current full value as a string. Use it for verification codes from SMS, email, or authenticator apps. Pair it with a timer or "Resend code" action for a complete verification UX.</Section.Text>
</Section>

<Section>
  <Section.Title>Accessibility</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>Built on <code>input-otp</code> with proper ARIA attributes for code input.</Section.ListItem>
    <Section.ListItem>Auto-focuses the first empty slot on mount for immediate input.</Section.ListItem>
    <Section.ListItem>Supports paste — users can paste the full code at once, and it distributes across slots.</Section.ListItem>
    <Section.ListItem><code>Tab</code> moves focus into and out of the OTP field as a single unit.</Section.ListItem>
    <Section.ListItem>Arrow keys move between individual slots for correction.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { OneTimePasswordField } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>Default</Section.Title>
  <ComponentExample name="otp-default">
    ```tsx
    "use client";

    import { OneTimePasswordField } from "@cocso-ui/react";
    import { useState } from "react";

    export default function OTPDefault() {
      const [value, setValue] = useState("");

      return (
        <div className="p-4">
          <OneTimePasswordField onValueChange={setValue} />
          {value && <p className="mt-2 text-sm text-neutral-500">Value: {value}</p>}
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
