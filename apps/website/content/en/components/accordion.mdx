---
title: Accordion
description: Vertically stacked interactive headings that expand or hide related content.
---

<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>Accordion is a compound component that follows the WAI-ARIA disclosure widget pattern and supports keyboard navigation by default. Each item can be expanded or collapsed independently. Use Accordion for collapsible content sections such as FAQs, settings panels, or detail sections where space is limited. Keep headers short and descriptive — they serve as the primary label for the hidden content. It supports both controlled (<code>value</code> + <code>onValueChange</code>) and uncontrolled (<code>defaultValue</code>) usage.</Section.Text>
</Section>

<Section>
  <Section.Title>Accessibility</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>Follows the WAI-ARIA Accordion (disclosure widget) pattern.</Section.ListItem>
    <Section.ListItem><code>Enter</code> or <code>Space</code> toggles the focused item open or closed.</Section.ListItem>
    <Section.ListItem><code>Arrow Down</code> moves focus to the next header. <code>Arrow Up</code> moves to the previous header.</Section.ListItem>
    <Section.ListItem><code>Home</code> moves focus to the first header. <code>End</code> moves to the last header.</Section.ListItem>
    <Section.ListItem>Each trigger has <code>aria-expanded</code> and <code>aria-controls</code> linking it to its content panel.</Section.ListItem>
  </Section.UnorderedList>
</Section>

  <SubHeading>Accordion (Root Container)</SubHeading>

  <PropsTable
    data={[
      { name: "defaultValue", type: "string | string[]", description: "Value of the item(s) expanded by default. Use for uncontrolled scenarios where the component manages its own open state." },
      { name: "value", type: "string | string[]", description: "Controlled expanded value. Accepts a single string or an array for multiple open items. Use together with `onValueChange`." },
      { name: "onValueChange", type: "(value: string | string[]) => void", description: "Callback invoked when items are expanded or collapsed. Receives the new value (string or array)." },

  <PropsTable
    data={[
      { name: "value", type: "string", required: true, description: "Unique string identifier for this accordion item. Must match the root's `value` or `defaultValue` to appear expanded." },

  <Section.Text>Renders an <code>&lt;h3&gt;</code> element wrapping the trigger. Accepts standard HTML heading props.</Section.Text>

  <PropsTable

    ]}
  />

  <Section.Text>An animated panel that reveals content. Accepts Base UI Accordion.Panel props.</Section.Text>
