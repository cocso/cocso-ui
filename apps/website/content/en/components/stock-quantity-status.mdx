<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>StockQuantityStatus is a domain-specific component that displays a color-coded stock level indicator alongside a label. It is used in pharmaceutical inventory management contexts to communicate whether stock is plentiful (sufficient), moderate (normal), or low (insufficient).</Section.Text>
  <Section.Text>This component is specific to pharmaceutical inventory management. Pair it with inventory tables or product detail views to provide an at-a-glance stock level assessment.</Section.Text>
</Section>

<Section>
  <Section.Title>Accessibility</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>The label text (<code>"sufficient"</code>, <code>"normal"</code>, <code>"insufficient"</code>) is read by screen readers, providing a text equivalent of the color indicator.</Section.ListItem>
    <Section.ListItem>Color is not the sole indicator — the text label always accompanies the color-coded icon, ensuring accessibility for color-blind users.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { StockQuantityStatus } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>Default</Section.Title>
  <Section.Text>Pass one of the three quantity values to render the corresponding color-coded indicator and label.</Section.Text>
  <ComponentExample name="stock-quantity-status-default">
    ```tsx
    import { StockQuantityStatus } from "@cocso-ui/react";

    export default function StockQuantityStatusDefault() {
      return (
        <div className="flex flex-wrap items-center gap-4 p-4">
          <StockQuantityStatus quantity="sufficient" />
          <StockQuantityStatus quantity="normal" />
          <StockQuantityStatus quantity="insufficient" />
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
