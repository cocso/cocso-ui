---
title: Tooltip
description: A floating label that provides supplementary information when an element is hovered or focused.
---

<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>
    Tooltip is a compound component that renders a short text label when the user hovers or focuses a trigger element. Wrapping multiple tooltips with a shared <code>Tooltip.Provider</code> lets you coordinate delay and open behavior across them.
  </Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>Composed with a compound API: <code>Tooltip.Provider</code>, <code>Tooltip</code>, <code>Tooltip.Trigger</code>, <code>Tooltip.Content</code>, <code>Tooltip.Arrow</code></Section.ListItem>
    <Section.ListItem>Wrapping a group of tooltips with <code>Tooltip.Provider</code> shares hover delay settings.</Section.ListItem>
    <Section.ListItem>Supports <code>side</code> and <code>align</code> positioning props on <code>Tooltip.Content</code>.</Section.ListItem>
    <Section.ListItem><code>Tooltip.Arrow</code> renders an arrow caret pointing toward the trigger.</Section.ListItem>
  </Section.UnorderedList>
  <Section.Text>Guidelines</Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>Use Tooltip for short, supplementary text labels — 1–2 sentences maximum.</Section.ListItem>
    <Section.ListItem>Tooltip content should not be essential for understanding the UI; it supplements the trigger label.</Section.ListItem>
    <Section.ListItem>For interactive content (links, buttons, forms), use Popover instead.</Section.ListItem>
    <Section.ListItem>Always wrap multiple nearby tooltips in a shared <code>Tooltip.Provider</code> to coordinate hover delays.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Accessibility</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>Tooltip content is exposed via <code>role="tooltip"</code> with the trigger linked via <code>aria-describedby</code>.</Section.ListItem>
    <Section.ListItem>Appears on both hover and focus, ensuring keyboard users can access the tooltip content.</Section.ListItem>
    <Section.ListItem>Disappears when the trigger loses hover or focus, or when <code>Escape</code> is pressed.</Section.ListItem>
    <Section.ListItem>Tooltip content should be supplementary — do not place critical information only in a tooltip, as some users may not discover it.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Tooltip } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>Default</Section.Title>
  <Section.Text>A tooltip that appears when hovering the trigger button.</Section.Text>
  <ComponentExample name="tooltip-default">
```tsx
"use client";

import { Button, Tooltip } from "@cocso-ui/react";

export default function TooltipDefault() {
  return (
    <div className="p-4">
      <Tooltip.Provider>
        <Tooltip>
          <Tooltip.Trigger render={<Button variant="secondary" />}>
            Hover me
          </Tooltip.Trigger>
          <Tooltip.Content>
            This is a tooltip
            <Tooltip.Arrow />
          </Tooltip.Content>
        </Tooltip>
      </Tooltip.Provider>
    </div>
  );
}
```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Positions</Section.Title>
  <Section.Text>Use the <code>side</code> prop on <code>Tooltip.Content</code> to control which side of the trigger the tooltip appears on.</Section.Text>
  <ComponentExample name="tooltip-positions">
```tsx
"use client";

import { Button, Tooltip } from "@cocso-ui/react";

export default function TooltipPositions() {
  return (
    <div className="flex flex-wrap gap-3 p-4">
      <Tooltip.Provider>
        {(["top", "bottom", "left", "right"] as const).map((side) => (
          <Tooltip key={side}>
            <Tooltip.Trigger render={<Button variant="outline" />}>{side}</Tooltip.Trigger>
            <Tooltip.Content side={side}>
              Tooltip on {side}
              <Tooltip.Arrow />
            </Tooltip.Content>
          </Tooltip>
        ))}
      </Tooltip.Provider>
    </div>
  );
}
```
  </ComponentExample>
</Section>
