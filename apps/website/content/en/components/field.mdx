---
title: Field
description: A form field wrapper that associates a label, description, and error message with a form control.
---

<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>Field wraps a form control with an accessible label and optional help text. It provides a description for additional guidance and an error message for validation feedback. Only one of description or error is shown at a time; error takes precedence. Note that Input and Select already integrate Field internally when a <code>label</code> prop is provided. Use Field directly when wrapping custom form controls or when you need a standalone label/error/description wrapper.</Section.Text>
</Section>

<Section>
  <Section.Title>Accessibility</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>Associates a <code>&lt;label&gt;</code> element with the child control via <code>htmlFor</code>/<code>id</code> for screen reader accessibility.</Section.ListItem>
    <Section.ListItem><code>error</code> and <code>description</code> text are linked to the child control via <code>aria-describedby</code>.</Section.ListItem>
    <Section.ListItem>Only one of <code>error</code> or <code>description</code> is rendered at a time — <code>error</code> takes precedence when both are provided.</Section.ListItem>
    <Section.ListItem>When <code>required</code> is <code>false</code>, an "(선택)" optional indicator appears next to the label.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Field } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>Default</Section.Title>
  <Section.Text>Pass a form control as a child. The <code>label</code> prop is required. Use <code>description</code> for help text and <code>error</code> for validation errors.</Section.Text>
  <ComponentExample name="field-default">
    ```tsx
    import { Field, Input } from "@cocso-ui/react";

    export default function FieldDefault() {
      return (
        <div className="flex w-full max-w-sm flex-col gap-4 p-4">
          <Field label="Email" description="We'll never share your email.">
            <Input placeholder="Enter your email" />
          </Field>
          <Field label="Password" required error="Password must be at least 8 characters.">
            <Input placeholder="Enter password" type="password" />
          </Field>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
