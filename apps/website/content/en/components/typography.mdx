---
title: Typography
description: A flexible text rendering component supporting custom, body, and heading modes.
---

<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>The Typography component operates in three modes controlled by the type prop. `custom` accepts arbitrary numeric font sizes, `body` is semantic body text with named size steps, and `heading` is semantic headings with responsive size steps. All modes support weight, lineHeight, and render props.</Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>Use <code>heading</code> type for page and section titles — <code>x-large</code> and <code>large</code> automatically scale up at the tablet breakpoint for responsive design.</Section.ListItem>
    <Section.ListItem>Use <code>body</code> for all standard prose and UI text.</Section.ListItem>
    <Section.ListItem>Use <code>custom</code> when a design specification requires a specific numeric size not available in the named scales.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Accessibility</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem><code>heading</code> type renders as <code>&lt;h2&gt;</code> by default. Use the <code>render</code> prop to set the correct heading level for your document outline (e.g., <code>render={`<h1 />`}</code>).</Section.ListItem>
    <Section.ListItem><code>body</code> and <code>custom</code> types render as <code>&lt;p&gt;</code> by default. Change the element with <code>render</code> when wrapping inline content (e.g., <code>render={`<span />`}</code>).</Section.ListItem>
    <Section.ListItem>Maintain a logical heading hierarchy (h1 → h2 → h3) regardless of visual size. A smaller heading can still be a higher level.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Typography } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>Heading</Section.Title>
  <Section.Text>Heading mode renders an h2 element by default. The x-large and large sizes are responsive, increasing in size at the tablet breakpoint.</Section.Text>
  <ComponentExample name="typography-heading">
    ```tsx
    import { Typography } from "@cocso-ui/react";

    export default function TypographyHeading() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Typography type="heading" size="x-large">X-Large Heading</Typography>
          <Typography type="heading" size="large">Large Heading</Typography>
          <Typography type="heading" size="medium">Medium Heading</Typography>
          <Typography type="heading" size="small">Small Heading</Typography>
          <Typography type="heading" size="x-small">X-Small Heading</Typography>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Body</Section.Title>
  <Section.Text>Body mode is the default. It renders a p element for all standard prose and UI text.</Section.Text>
  <ComponentExample name="typography-body">
    ```tsx
    import { Typography } from "@cocso-ui/react";

    export default function TypographyBody() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Typography type="body" size="large">Large body text for prominent content.</Typography>
          <Typography type="body" size="medium">Medium body text for standard content.</Typography>
          <Typography type="body" size="small">Small body text for secondary content.</Typography>
          <Typography type="body" size="x-small">X-Small body text for captions.</Typography>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Custom</Section.Title>
  <Section.Text>Custom mode accepts an arbitrary numeric font size from the design token scale, or a responsive object with base, tablet, and desktop keys.</Section.Text>
  <ComponentExample name="typography-custom">
    ```tsx
    import { Typography } from "@cocso-ui/react";

    export default function TypographyCustom() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Typography type="custom" size={24} weight="bold">Custom size 24, bold</Typography>
          <Typography type="custom" size={16} weight="medium">Custom size 16, medium</Typography>
          <Typography type="custom" size={12} weight="normal" lineHeight="relaxed">Custom size 12, relaxed line height</Typography>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
