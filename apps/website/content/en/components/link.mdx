---
title: Link
description: A styled anchor element supporting variants, sizes, and external link indicators.
---

<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>Link renders a styled anchor element built on top of Typography. It supports three visual variants for different contexts, four size options, and an optional external link indicator icon.</Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>Use <code>inline</code> for standard hyperlinks within body text.</Section.ListItem>
    <Section.ListItem>Use <code>current</code> to indicate the active page in a navigation context (it inherits the surrounding text color).</Section.ListItem>
    <Section.ListItem>Use <code>plain</code> for interactive text that should not appear as a traditional link.</Section.ListItem>
    <Section.ListItem>For external links, always include <code>Link.ExternalIcon</code> and set <code>target="_blank"</code> with <code>rel="noopener noreferrer"</code>.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Accessibility</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>Renders as a native <code>&lt;a&gt;</code> element with built-in keyboard support. <code>Enter</code> activates the link.</Section.ListItem>
    <Section.ListItem>For links that open in a new tab, combine <code>target="_blank"</code> with <code>rel="noopener noreferrer"</code> to prevent security issues.</Section.ListItem>
    <Section.ListItem><code>Link.ExternalIcon</code> provides a visual indicator for external links. Consider also adding a screen-reader-only label like <code>"(opens in a new tab)"</code> for clarity.</Section.ListItem>
    <Section.ListItem>Use the <code>render</code> prop to integrate with client-side routing (e.g., Next.js <code>Link</code>) while preserving the styled appearance.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Link } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>Variant</Section.Title>
  <Section.Text>Three variants control color and underline behavior. <code>inline</code> is a standard hyperlink, <code>current</code> inherits the surrounding text color, and <code>plain</code> is a link without underline.</Section.Text>
  <ComponentExample name="link-default">
    ```tsx
    import { Link } from "@cocso-ui/react";

    export default function LinkDefault() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Link href="#" variant="inline">Inline link</Link>
          <Link href="#" variant="current">Current link</Link>
          <Link href="#" variant="plain">Plain link</Link>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Size</Section.Title>
  <Section.Text>The size prop maps to the Typography body size scale: <code>large</code> (18px), <code>medium</code> (16px), <code>small</code> (14px), <code>x-small</code> (12px).</Section.Text>
  <ComponentExample name="link-sizes">
    ```tsx
    import { Link } from "@cocso-ui/react";

    export default function LinkSizes() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Link href="#" size="large">Large link</Link>
          <Link href="#" size="medium">Medium link</Link>
          <Link href="#" size="small">Small link</Link>
          <Link href="#" size="x-small">X-Small link</Link>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>External</Section.Title>
  <Section.Text>Place Link.ExternalIcon inside the link content to display an external link icon. The <code>indicator</code> prop lets you explicitly control the underline indicator.</Section.Text>
  <ComponentExample name="link-external">
    ```tsx
    import { Link } from "@cocso-ui/react";

    export default function LinkExternal() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Link href="https://example.com" indicator target="_blank" rel="noopener noreferrer">
            External link <Link.ExternalIcon />
          </Link>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
