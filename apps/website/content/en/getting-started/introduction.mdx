<Section>
  <Section.Title>Background</Section.Title>
  <Section.Text>COCSO Inc. operates web services for a wide range of user types — general users, business administrators, pharmaceutical company administrators, and super administrators. As the service scope expanded, components, icons, and styles were being defined redundantly, and inconsistencies between service interfaces grew, creating the need for a unified design system.</Section.Text>
</Section>

<Section>
  <Section.Title>Features</Section.Title>
  <Section.Text>COCSO-UI is based on <Link href="https://www.krds.go.kr/html/site/index.html" target="_blank" rel="noopener noreferrer" type="custom" size="current">KRDS<LinkExternalIcon /></Link>, with some colors and sizes adjusted. Extensibility has been kept open to allow customization.</Section.Text>
  <Section.Text>Key features are as follows.</Section.Text>

  <Section.OrderedList>
    <Section.ListItem>Styles are encapsulated using CSS Modules. With no external CSS library dependency, it is compatible with any project.</Section.ListItem>
    <Section.ListItem>Built on Base UI, it follows the <Link href="https://www.w3.org/WAI/ARIA/apg/" target="_blank" rel="noopener noreferrer" type="custom" size="current">WAI-ARIA<LinkExternalIcon /></Link> web accessibility standards.</Section.ListItem>
    <Section.ListItem>Written in TypeScript, it supports reliable type inference.</Section.ListItem>
  </Section.OrderedList>
</Section>
