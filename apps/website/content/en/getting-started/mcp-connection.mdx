<Section>
  <Section.Title>Client Setup by Tool</Section.Title>
  <Tabs items={["Claude Code", "Codex", "OpenCode"]}>
    <Tab value="Claude Code">
      <Section.Text>Claude Code supports direct MCP remote servers over HTTP. Add this to project `.mcp.json` or user `~/.claude.json`.</Section.Text>

      ```json
      {
        "mcpServers": {
          "cocso-ui": {
            "type": "http",
            "url": "https://www.cocso-ui.com/api/mcp"
          }
        }
      }
      ```

      <Section.Text>Note: Claude Desktop uses `claude_desktop_config.json` with stdio-style server entries. Remote MCP in Desktop is handled through its extension flow, so direct HTTP setup differs from Claude Code.</Section.Text>
    </Tab>
    <Tab value="Codex">
      <Section.Text>Codex uses TOML config. Add this to `~/.codex/config.toml` (or `.codex/config.toml` for project scope).</Section.Text>

      ```toml
      [mcp_servers.cocso_ui]
      url = "https://www.cocso-ui.com/api/mcp"
      ```
    </Tab>
    <Tab value="OpenCode">
      <Section.Text>OpenCode supports remote MCP directly. Add this to `~/.config/opencode/opencode.json` (or `opencode.json` in your project).</Section.Text>

      ```json
      {
        "$schema": "https://opencode.ai/config.json",
        "mcp": {
          "cocso-ui": {
            "type": "remote",
            "url": "https://www.cocso-ui.com/api/mcp",
            "enabled": true
          }
        }
      }
      ```
    </Tab>
  </Tabs>
</Section>

<Section>
  <Section.Title>stdio-only clients</Section.Title>
  <Section.Text>If a client only supports stdio transport, bridge to the hosted endpoint with <Link href="https://www.npmjs.com/package/mcp-remote" target="_blank" rel="noopener noreferrer" type="custom" size="current">mcp-remote<LinkExternalIcon /></Link>.</Section.Text>

  ```json
  {
    "cocso-ui": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://www.cocso-ui.com/api/mcp"]
    }
  }
  ```
</Section>

<Section>
  <Section.Title>Recommended Workflow</Section.Title>
  <Section.OrderedList>
    <Section.ListItem>Call `cocso_ui_mcp_search_components` first with your page intent.</Section.ListItem>
    <Section.ListItem>Use `cocso_ui_mcp_get_component_spec` for selected components before coding.</Section.ListItem>
    <Section.ListItem>Import reusable UI from `@cocso-ui/react` instead of generating duplicate local components.</Section.ListItem>
  </Section.OrderedList>
</Section>
