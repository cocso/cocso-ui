title: Tab

<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>Tab is a compound component built on Base UI's Tabs primitive. It provides a keyboard-accessible tab interface following the WAI-ARIA pattern. Content panels are linked to their corresponding triggers via a shared `value` prop.</Section.Text>
</Section>

<Section>
  <Section.Title>Default</Section.Title>
  <Section.Text>The default tab renders a list of triggers and linked content panels. Use `defaultValue` for uncontrolled usage and `value` with `onValueChange` for controlled usage.</Section.Text>
  <ComponentExample name="tab-default" />
</Section>
