---
title: Field
description: A form field wrapper that associates a label, description, and error message with a form control.
---

<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>Field wraps a form control with an accessible label and optional help text. It provides a description for additional guidance and an error message for validation feedback. Only one of description or error is shown at a time; error takes precedence.</Section.Text>
</Section>

<Section>
  <Section.Title>Default</Section.Title>
  <Section.Text>Pass a form control as a child. The label prop is required. Use description for help text and error for validation errors.</Section.Text>
  <ComponentExample name="field-default">
    ```tsx
    import { Field, Input } from "@cocso-ui/react";

    export default function FieldDefault() {
      return (
        <div className="flex w-full max-w-sm flex-col gap-4 p-4">
          <Field label="Email" description="We'll never share your email.">
            <Input placeholder="Enter your email" />
          </Field>
          <Field label="Password" required error="Password must be at least 8 characters.">
            <Input placeholder="Enter password" type="password" />
          </Field>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>API Reference</Section.Title>
  <PropsTable data={[
    { name: "label", type: "string", required: true, description: "Label text rendered above the form control." },
    { name: "children", type: "ReactNode", required: true, description: "The form control element (e.g. Input, Select)." },
    { name: "description", type: "string", description: "Help text displayed below the control when there is no error." },
    { name: "error", type: "string", description: "Error message displayed below the control. When set, replaces description." },
    { name: "htmlFor", type: "string", description: "The id of the form control used for the label's for attribute. Improves accessibility when the child is not automatically associated." },
    { name: "required", type: "boolean", description: "When false, an optional indicator appears next to the label." },
  ]} />
</Section>
