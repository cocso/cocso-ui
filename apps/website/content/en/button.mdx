---
title: Button
description: A clickable element that triggers an action or event.
---

<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>
    The Button component triggers actions or navigation. It supports a variety of visual variants, sizes, shapes, and states including loading and disabled. Use it as a primary call-to-action, a secondary control, or an icon-only affordance.
  </Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>Use <code>primary</code> for the main call-to-action on a page — limit to one per section for visual clarity.</Section.ListItem>
    <Section.ListItem>Use <code>secondary</code> or <code>outline</code> for supporting actions, and <code>ghost</code> for inline or tertiary controls.</Section.ListItem>
    <Section.ListItem>Reserve semantic variants (<code>success</code>, <code>error</code>, <code>warning</code>, <code>info</code>) for actions with explicit status outcomes.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Accessibility</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>Renders as a native <code>&lt;button&gt;</code> element with built-in keyboard support.</Section.ListItem>
    <Section.ListItem><code>Space</code> or <code>Enter</code> activates the button.</Section.ListItem>
    <Section.ListItem>When <code>loading</code> is set, the button shows a spinner overlay with <code>role="status"</code> and becomes non-interactive.</Section.ListItem>
    <Section.ListItem>When <code>disabled</code>, the button is removed from the tab order and cannot receive focus.</Section.ListItem>
    <Section.ListItem>Use the <code>render</code> prop to render as a link (<code>&lt;a&gt;</code>) when the button navigates to a URL, preserving semantic meaning.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Button } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>Variant</Section.Title>
  <Section.Text>
    Use variants to communicate the intent of an action. `primary` is the default high-emphasis style; use `secondary`, `outline`, or `ghost` for lower emphasis. State variants (`success`, `error`, `warning`, `info`) convey semantic meaning.
  </Section.Text>
  <ComponentExample name="button-variants">
    ```tsx
    import { Button } from "@cocso-ui/react";

    export default function ButtonVariants() {
      return (
        <div className="flex flex-wrap items-center gap-3 p-4">
          <Button variant="primary">Primary</Button>
          <Button variant="secondary">Secondary</Button>
          <Button variant="outline">Outline</Button>
          <Button variant="ghost">Ghost</Button>
          <Button variant="success">Success</Button>
          <Button variant="error">Error</Button>
          <Button variant="warning">Warning</Button>
          <Button variant="info">Info</Button>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Size</Section.Title>
  <Section.Text>
    Four sizes are available. Choose based on context and surrounding content density.
  </Section.Text>
  <ComponentExample name="button-sizes">
    ```tsx
    import { Button } from "@cocso-ui/react";

    export default function ButtonSizes() {
      return (
        <div className="flex flex-wrap items-center gap-3 p-4">
          <Button size="large">Large</Button>
          <Button size="medium">Medium</Button>
          <Button size="small">Small</Button>
          <Button size="x-small">X-Small</Button>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Shape</Section.Title>
  <Section.Text>
    Use <code>square</code> for standard rectangular buttons, <code>rounded</code> for a pill style, and combine <code>circle</code> with <code>svgOnly</code> for icon-only circular buttons.
  </Section.Text>
  <ComponentExample name="button-shapes">
    ```tsx
    import { Button } from "@cocso-ui/react";
    import { PlusIcon } from "@cocso-ui/react-icons";

    export default function ButtonShapes() {
      return (
        <div className="flex flex-wrap items-center gap-3 p-4">
          <Button shape="square">Square</Button>
          <Button shape="rounded">Rounded</Button>
          <Button shape="circle" svgOnly><PlusIcon size={18} /></Button>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Loading</Section.Title>
  <Section.Text>
    Set <code>loading</code> to display a spinner and disable interaction while an async operation is in progress.
  </Section.Text>
  <ComponentExample name="button-loading">
    ```tsx
    import { Button } from "@cocso-ui/react";

    export default function ButtonLoading() {
      return (
        <div className="flex flex-wrap items-center gap-3 p-4">
          <Button loading variant="primary">Loading</Button>
          <Button loading variant="secondary">Loading</Button>
          <Button loading variant="outline">Loading</Button>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Prefix & Suffix</Section.Title>
  <Section.Text>
    Use <code>prefix</code> and <code>suffix</code> to place an icon or element before or after the button label.
  </Section.Text>
  <ComponentExample name="button-prefix-suffix">
    ```tsx
    import { Button } from "@cocso-ui/react";
    import { ArrowForwardIcon, PlusIcon } from "@cocso-ui/react-icons";

    export default function ButtonPrefixSuffix() {
      return (
        <div className="flex flex-wrap items-center gap-3 p-4">
          <Button prefix={<PlusIcon size={16} />}>Add item</Button>
          <Button suffix={<ArrowForwardIcon size={16} />}>Continue</Button>
          <Button prefix={<PlusIcon size={16} />} suffix={<ArrowForwardIcon size={16} />}>Both</Button>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Disabled</Section.Title>
  <Section.Text>
    Set <code>disabled</code> to prevent interaction. Disabled buttons appear visually dimmed and cannot be focused.
  </Section.Text>
  <ComponentExample name="button-variants">
    ```tsx
    import { Button } from "@cocso-ui/react";

    export default function ButtonDisabled() {
      return (
        <div className="flex flex-wrap items-center gap-3 p-4">
          <Button disabled variant="primary">Disabled Primary</Button>
          <Button disabled variant="outline">Disabled Outline</Button>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
