<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>StockQuantityStatus is a domain-specific component that displays a color-coded stock level indicator alongside a label. It is used in pharmaceutical inventory management contexts to communicate whether stock is plentiful (여유), moderate (보통), or low (부족).</Section.Text>
  <Section.Text>This component is specific to pharmaceutical inventory management. Pair it with inventory tables or product detail views to provide an at-a-glance stock level assessment.</Section.Text>
</Section>

<Section>
  <Section.Title>Accessibility</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>The label text (<code>"여유"</code>, <code>"보통"</code>, <code>"부족"</code>) is read by screen readers, providing a text equivalent of the color indicator.</Section.ListItem>
    <Section.ListItem>Color is not the sole indicator — the text label always accompanies the color-coded icon, ensuring accessibility for color-blind users.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { StockQuantityStatus } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>Default</Section.Title>

  <ComponentExample name="stock-quantity-status-default">
    ```tsx

    export default function StockQuantityStatusDefault() {
      return (
        <div className="flex flex-wrap items-center gap-4 p-4">

          <StockQuantityStatus quantity="보통" />
          <StockQuantityStatus quantity="부족" />
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
