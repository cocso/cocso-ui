---
title: Accordion
description: Vertically stacked interactive headings that expand or hide related content.
---

<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>Accordion is a compound component built on Base UI's Accordion primitive. It follows the WAI-ARIA disclosure widget pattern and supports keyboard navigation by default. Each item can be expanded or collapsed independently.</Section.Text>
</Section>

<Section>
  <Section.Title>Default</Section.Title>
  <Section.Text>The default accordion renders each item with a chevron indicator that rotates when opened.</Section.Text>
  <ComponentExample name="accordion-default" />
</Section>

<Section>
  <Section.Title>Hide Chevron</Section.Title>
  <Section.Text>Set `chevron={false}` on `Accordion.Trigger` to hide the chevron icon.</Section.Text>
  <ComponentExample name="accordion-no-chevron" />
</Section>

  <SubHeading>Accordion.Header</SubHeading>

  <PropsTable

  />

  <SubHeading>Accordion.Content</SubHeading>
