<Section>
  <Section.Title>Overview</Section.Title>
  <Section.Text>COCSO UI uses Pretendard Variable as the default sans-serif font and Geist Mono for code. The typography system is based on a fixed numeric scale that maps directly to CSS custom properties with the `--cocso-*` prefix.</Section.Text>
</Section>
