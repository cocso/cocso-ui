<Section>
  <Section.Title>배경</Section.Title>
  <Section.Text>주식회사 코쏘는 일반 사용자, 비즈니스 관리자, 제약사 관리자, 최고 관리자 등 다양한 유형의 사용자를 대상으로 웹 서비스를 운영합니다. 서비스 범위가 확장되면서 컴포넌트, 아이콘, 스타일이 중복 정의되는 문제가 생겼고, 서비스 간 인터페이스 불일치가 심화되어 일관된 디자인 시스템이 필요해졌습니다.</Section.Text>
</Section>

<Section>
  <Section.Title>특징</Section.Title>
  <Section.Text>COCSO-UI는 <Link href="https://www.krds.go.kr/html/site/index.html" target="_blank" rel="noopener noreferrer" type="custom" size={15}>KRDS<LinkExternalIcon /></Link>를 기반으로 하되, 일부 색상과 크기를 조정했습니다. 커스텀이 가능하도록 확장성을 열어두었습니다.</Section.Text>
  <Section.Text>주요 특징은 다음과 같습니다.</Section.Text>

  <Section.OrderedList>
    <Section.ListItem>CSS Modules를 사용하여 스타일이 캡슐화되어 있습니다. 별도의 CSS 라이브러리 의존성이 없어 모든 프로젝트와 호환됩니다.</Section.ListItem>
    <Section.ListItem>Base UI 기반으로 제작되어 <Link href="https://www.w3.org/WAI/ARIA/apg/" target="_blank" rel="noopener noreferrer" type="custom" size={15}>WAI-ARIA<LinkExternalIcon /></Link> 웹 접근성 표준을 준수합니다.</Section.ListItem>
    <Section.ListItem>TypeScript로 작성되어 안정적인 타입 추론을 지원합니다.</Section.ListItem>
  </Section.OrderedList>
</Section>
