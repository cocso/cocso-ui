---
title: Link
description: 변형, 크기, 외부 링크를 지원하는 스타일이 적용된 앵커 요소입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>Link는 Typography를 기반으로 만들어진 스타일이 적용된 앵커 요소를 렌더링합니다. 다양한 맥락에 맞는 세 가지 시각적 변형, 네 가지 크기 옵션, 선택적 외부 링크 표시기 아이콘을 지원합니다.</Section.Text>
</Section>

<Section>
  <Section.Title>Variant</Section.Title>
  <Section.Text>세 가지 변형이 색상과 밑줄 동작을 제어합니다. inline은 표준 하이퍼링크, current는 주변 텍스트 색상 상속, plain은 자동 밑줄 없는 링크입니다.</Section.Text>
  <ComponentExample name="link-default">
    ```tsx
    import { Link } from "@cocso-ui/react";

    export default function LinkDefault() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Link href="#" variant="inline">Inline link</Link>
          <Link href="#" variant="current">Current link</Link>
          <Link href="#" variant="plain">Plain link</Link>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>크기</Section.Title>
  <Section.Text>size prop은 Typography body 크기 스케일에 매핑됩니다: large (18px), medium (16px), small (14px), x-small (12px).</Section.Text>
  <ComponentExample name="link-sizes">
    ```tsx
    import { Link } from "@cocso-ui/react";

    export default function LinkSizes() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Link href="#" size="large">Large link</Link>
          <Link href="#" size="medium">Medium link</Link>
          <Link href="#" size="small">Small link</Link>
          <Link href="#" size="x-small">X-Small link</Link>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>외부 링크</Section.Title>
  <Section.Text>링크 콘텐츠 내부에 Link.ExternalIcon을 사용하여 외부 링크 아이콘을 표시하세요. indicator prop으로 밑줄 표시기 표시 여부를 명시적으로 제어할 수 있습니다.</Section.Text>
  <ComponentExample name="link-external">
    ```tsx
    import { Link } from "@cocso-ui/react";

    export default function LinkExternal() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Link href="https://example.com" indicator target="_blank" rel="noopener noreferrer">
            External link <Link.ExternalIcon />
          </Link>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
