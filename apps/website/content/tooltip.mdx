---
title: Tooltip
description: 요소에 호버하거나 포커스할 때 보조 정보를 제공하는 플로팅 레이블입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>
    Tooltip은 Base UI의 Tooltip 프리미티브 기반으로 구성된 복합 컴포넌트입니다. 사용자가 트리거 요소에 호버하거나 포커스할 때 짧은 텍스트 레이블을 렌더링합니다. 여러 툴팁을 공유된 <code>Tooltip.Provider</code>로 감싸면 지연 및 열기 동작을 일괄 조정할 수 있습니다.
  </Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>복합 API: <code>Tooltip.Provider</code>, <code>Tooltip</code>, <code>Tooltip.Trigger</code>, <code>Tooltip.Content</code>, <code>Tooltip.Arrow</code></Section.ListItem>
    <Section.ListItem>툴팁 그룹을 <code>Tooltip.Provider</code>로 감싸면 호버 지연 설정을 공유할 수 있습니다.</Section.ListItem>
    <Section.ListItem><code>Tooltip.Content</code>에서 <code>side</code> 및 <code>align</code> 위치 지정 props를 지원합니다.</Section.ListItem>
    <Section.ListItem><code>Tooltip.Arrow</code>는 트리거를 향하는 방향성 캐럿을 렌더링합니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <Section.Text>트리거 버튼에 호버할 때 나타나는 툴팁입니다.</Section.Text>
  <ComponentExample name="tooltip-default">
```tsx
"use client";

import { Button, Tooltip } from "@cocso-ui/react";

export default function TooltipDefault() {
  return (
    <div className="p-4">
      <Tooltip.Provider>
        <Tooltip>
          <Tooltip.Trigger render={<Button variant="secondary" />}>
            Hover me
          </Tooltip.Trigger>
          <Tooltip.Content>
            This is a tooltip
            <Tooltip.Arrow />
          </Tooltip.Content>
        </Tooltip>
      </Tooltip.Provider>
    </div>
  );
}
```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>위치</Section.Title>
  <Section.Text><code>Tooltip.Content</code>의 <code>side</code> prop을 사용해 툴팁이 트리거의 어느 방향에 표시될지 제어합니다.</Section.Text>
  <ComponentExample name="tooltip-positions">
```tsx
"use client";

import { Button, Tooltip } from "@cocso-ui/react";

export default function TooltipPositions() {
  return (
    <div className="flex flex-wrap gap-3 p-4">
      <Tooltip.Provider>
        {(["top", "bottom", "left", "right"] as const).map((side) => (
          <Tooltip key={side}>
            <Tooltip.Trigger render={<Button variant="outline" />}>{side}</Tooltip.Trigger>
            <Tooltip.Content side={side}>
              Tooltip on {side}
              <Tooltip.Arrow />
            </Tooltip.Content>
          </Tooltip>
        ))}
      </Tooltip.Provider>
    </div>
  );
}
```
  </ComponentExample>
</Section>
