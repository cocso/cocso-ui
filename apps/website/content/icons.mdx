---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>COCSO UI는 두 가지 카테고리에 걸쳐 83개의 아이콘을 제공합니다. 내비게이션, 액션, 상태, 도메인별 개념을 다루는 68개의 시맨틱 아이콘과 15개의 브랜드 아이콘으로 구성됩니다. 모든 아이콘은 `@cocso-ui/react-icons` 패키지에서 React 컴포넌트로 배포되며 인라인 SVG로 렌더링됩니다.</Section.Text>
</Section>

<Section>
  <Section.Title>사용법</Section.Title>
  <Section.Text>패키지에서 이름으로 원하는 아이콘을 가져옵니다. 각 아이콘은 `Icon`으로 끝나는 named export입니다.</Section.Text>

```tsx
import { SearchIcon } from "@cocso-ui/react-icons";

export default function Example() {
  return <SearchIcon size={20} />;
}
```
</Section>

<Section>
  <Section.Title>Props</Section.Title>
  <Section.Text>아이콘 컴포넌트는 모든 표준 SVG 요소 속성에 더해 다음 props를 허용합니다.</Section.Text>
  <TokenTable
    data={[
      { name: "size", token: "number", value: "16 (default)" },
      { name: "width", token: "number", value: "Falls back to size" },
      { name: "height", token: "number", value: "Falls back to size" },
    ]}
  />
  <Section.Text>아이콘은 기본적으로 보조 기술에서 숨겨지도록 자동으로 `aria-hidden="true"`로 표시됩니다. 아이콘이 단독으로 의미를 전달할 경우 사용 위치에서 접근 가능한 레이블을 제공하세요.</Section.Text>
</Section>

<Section>
  <Section.Title>갤러리</Section.Title>
