---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>DayPicker는 인라인 달력이 포함된 Dropdown을 여는 트리거 요소를 렌더링합니다. 제어 컴포넌트로, value와 onValueChange를 전달하여 선택된 날짜를 관리합니다. 달력은 기본적으로 한국어로 현지화됩니다.</Section.Text>
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <Section.Text>value와 onValueChange props로 선택된 날짜를 제어하세요. 커스텀 트리거가 제공되지 않으면 기본 버튼으로 렌더링됩니다.</Section.Text>
  <ComponentExample name="day-picker-default">
    ```tsx
    "use client";

    import { DayPicker } from "@cocso-ui/react";
    import { useState } from "react";

    export default function DayPickerDefault() {
      const [value, setValue] = useState<Date | null>(null);

      return (
        <div className="p-4">
          <DayPicker onValueChange={setValue} value={value ?? undefined} />
          {value && <p className="mt-2 text-sm text-neutral-500">Selected: {value.toLocaleDateString()}</p>}
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
