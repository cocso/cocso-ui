---
title: Badge
description: 상태, 카테고리, 수량을 강조하는 소형 레이블입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>Badge는 상태, 카테고리, 또는 숫자 수량을 전달하는 작은 인라인 요소입니다. 시맨틱 색상 변형, 세 가지 크기, 세 가지 모양 옵션을 지원하여 가장 일반적인 레이블링 패턴을 커버합니다.</Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>중립적인 카테고리 레이블이나 태그에는 <code>primary</code>와 <code>secondary</code>를 사용합니다.</Section.ListItem>
    <Section.ListItem>시맨틱 변형(<code>success</code>, <code>error</code>, <code>warning</code>, <code>info</code>)은 상태 인디케이터에 사용합니다.</Section.ListItem>
    <Section.ListItem><code>outline</code> 변형은 낮은 강조의 미묘한 레이블에 사용합니다.</Section.ListItem>
    <Section.ListItem>Badge는 비상호작용 표시 요소입니다. 클릭 가능한 레이블에는 적절한 변형의 Button을 사용합니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>Badge는 상호작용 역할 없이 <code>&lt;div&gt;</code>로 렌더링됩니다. 스크린 리더는 텍스트 콘텐츠를 자연스럽게 읽습니다.</Section.ListItem>
    <Section.ListItem>배지 콘텐츠가 동적으로 업데이트될 때(예: 알림 카운트)는 보조 기술에 변경 사항을 알리기 위해 <code>role="status"</code> 추가를 고려하세요.</Section.ListItem>
    <Section.ListItem>특히 <code>outline</code> 변형에서 배지 텍스트와 배경 간의 충분한 색상 대비를 확보하세요.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Badge } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>크기</Section.Title>
  <Section.Text>세 가지 크기를 사용할 수 있습니다. <code>large</code>, <code>medium</code>, <code>small</code>이며 기본값은 <code>medium</code>입니다.</Section.Text>
  <ComponentExample name="badge-sizes" />
</Section>
