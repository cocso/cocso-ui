<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>COCSO UI는 기본 sans-serif 폰트로 Pretendard Variable을, 코드용으로 Geist Mono를 사용합니다. 타이포그래피 시스템은 `--cocso-*` 접두사를 가진 CSS 커스텀 속성에 직접 매핑되는 고정 숫자 스케일을 기반으로 합니다.</Section.Text>
</Section>

<Section>
  <Section.Title>Font Size</Section.Title>
  <Section.Text>크기 스케일은 10px에서 60px까지 15단계로 구성됩니다. 각 단계는 아래에 실제 렌더링된 크기로 표시됩니다.</Section.Text>
  <ComponentExample name="typography-scale" />
</Section>

<Section>
  <Section.Title>Font Weight</Section.Title>
  <Section.Text>9가지 굵기 변형이 thin(100)부터 black(900)까지 전체 범위를 아우르며, 시각적 계층 구조와 강조를 정밀하게 제어할 수 있습니다.</Section.Text>
  <ComponentExample name="typography-weight" />
</Section>
