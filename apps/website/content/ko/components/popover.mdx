---
title: Popover
description: 트리거 요소에 고정된 플로팅 패널로, Base UI Popover 프리미티브 기반입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>
    Popover는 Dialog와 달리 비모달 방식으로 동작하여 페이지의 나머지 부분과의 상호작용을 차단하지 않는 복합 컴포넌트입니다. 보조 정보, 인라인 폼, 또는 내용이 풍부한 툴팁이 필요할 때 사용하세요.
  </Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>복합 API로 구성됩니다. <code>Popover</code>, <code>Popover.Trigger</code>, <code>Popover.Content</code>, <code>Popover.Arrow</code></Section.ListItem>
    <Section.ListItem>비모달 방식으로 동작하며, 외부를 클릭하면 상호작용을 차단하지 않고 팝오버가 닫힙니다.</Section.ListItem>
    <Section.ListItem><code>side</code> 및 <code>align</code> 위치 지정 props를 지원합니다.</Section.ListItem>
    <Section.ListItem><code>Popover.Arrow</code>는 트리거를 향하는 화살표 캐럿을 렌더링합니다.</Section.ListItem>
    <Section.ListItem>제어형(<code>open</code> + <code>onOpenChange</code>)과 비제어형(<code>defaultOpen</code>) 사용을 모두 지원합니다.</Section.ListItem>
  </Section.UnorderedList>
  <Section.Text className="font-semibold">가이드라인</Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>트리거에 고정된 보조 정보, 인라인 폼, 또는 풍부한 인터랙티브 콘텐츠에 Popover를 사용합니다.</Section.ListItem>
    <Section.ListItem>짧은 텍스트 전용 힌트에는 Tooltip을 사용합니다 (더 가볍고 hover 전용입니다).</Section.ListItem>
    <Section.ListItem>사용자의 주의가 반드시 필요한 콘텐츠에는 Dialog를 사용합니다 (모달, 포커스 트래핑).</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>기본적으로 비모달 방식입니다. 포커스를 가두거나 페이지의 나머지 부분과의 상호작용을 차단하지 않습니다.</Section.ListItem>
    <Section.ListItem>외부를 클릭하거나 <code>Escape</code>를 누르면 팝오버가 닫힙니다.</Section.ListItem>
    <Section.ListItem>트리거 요소에 <code>aria-expanded</code>가 적용되어 팝오버의 열림 상태를 전달합니다.</Section.ListItem>
    <Section.ListItem>팝오버 콘텐츠 내에서 자연스러운 탭 순서에 따라 포커스가 관리됩니다. 폼과 버튼 같은 인터랙티브 요소와 함께 사용하세요.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Popover } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <Section.Text>제목, 설명 텍스트, 화살표 인디케이터가 있는 팝오버입니다.</Section.Text>
  <ComponentExample name="popover-default">
```tsx
"use client";

import { Button, Popover, Typography } from "@cocso-ui/react";

export default function PopoverDefault() {
  return (
    <div className="p-4">
      <Popover>
        <Popover.Trigger render={<Button variant="secondary" />}>Open Popover</Popover.Trigger>
        <Popover.Content>
          <div className="flex flex-col gap-2 p-3">
            <Typography size={14} weight="semibold">Popover Title</Typography>
            <Typography size={13}>This is the popover content. It can contain any elements.</Typography>
          </div>
          <Popover.Arrow />
        </Popover.Content>
      </Popover>
    </div>
  );
}
```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>위치</Section.Title>
  <Section.Text><code>Popover.Content</code>의 <code>side</code> prop을 사용해 팝오버가 트리거의 어느 방향에 표시될지 제어합니다.</Section.Text>
  <ComponentExample name="popover-positions">
```tsx
"use client";

import { Button, Popover, Typography } from "@cocso-ui/react";

export default function PopoverPositions() {
  return (
    <div className="flex flex-wrap gap-3 p-4">
      {(["top", "bottom", "left", "right"] as const).map((side) => (
        <Popover key={side}>
          <Popover.Trigger render={<Button variant="outline" />}>{side}</Popover.Trigger>
          <Popover.Content side={side}>
            <div className="p-3">
              <Typography size={13}>Popover on {side}</Typography>
            </div>
            <Popover.Arrow />
          </Popover.Content>
        </Popover>
      ))}
    </div>
  );
}
```
  </ComponentExample>
</Section>
