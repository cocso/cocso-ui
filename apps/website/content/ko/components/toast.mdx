---
title: Toast
description: 화면 가장자리에 나타나는 간결한 비모달 알림입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>
    Toast는 COCSO UI 스타일이 적용된 <a href="https://sonner.emilkowal.ski/" target="_blank" rel="noopener noreferrer">Sonner</a> 기반 알림 컴포넌트입니다. 애플리케이션 어디서든 호출할 수 있는 명령형 <code>toast()</code> 함수와, 토스트 뷰포트를 마운트하는 <code>Toaster</code> 컴포넌트를 제공합니다.
  </Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>명령형 API: 어떤 이벤트 핸들러에서든 <code>toast()</code>, <code>toast.success()</code>, <code>toast.error()</code> 등을 호출할 수 있습니다.</Section.ListItem>
    <Section.ListItem>루트 레이아웃에 <code>&lt;Toaster /&gt;</code>를 한 번 배치하면 토스트 렌더링이 활성화됩니다.</Section.ListItem>
    <Section.ListItem>토스트는 쌓이며 설정 가능한 지속 시간 후 자동으로 닫힙니다.</Section.ListItem>
    <Section.ListItem><code>success</code>, <code>error</code>, <code>info</code>, <code>warning</code> 시맨틱 변형을 지원합니다.</Section.ListItem>
  </Section.UnorderedList>
  <Section.Text className="font-semibold">가이드라인</Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>완료된 액션에 대한 일시적이고 비차단 방식의 피드백에 Toast를 사용합니다. 예: "저장되었습니다", "링크가 복사되었습니다", "항목이 삭제되었습니다".</Section.ListItem>
    <Section.ListItem>사용자의 액션이나 결정이 필요한 오류에는 Dialog를 사용합니다.</Section.ListItem>
    <Section.ListItem>토스트 메시지는 한눈에 읽을 수 있도록 간결하게 (100자 이하) 유지합니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>토스트는 <code>aria-live="polite"</code> 영역에 렌더링되어 스크린 리더가 현재 작업을 방해하지 않고 알림을 읽어줍니다.</Section.ListItem>
    <Section.ListItem>시맨틱 변형 (<code>success</code>, <code>error</code> 등)은 시각적 표시와 읽어주는 컨텍스트를 함께 제공합니다.</Section.ListItem>
    <Section.ListItem>사용자가 자동 닫힘 전에 토스트를 수동으로 닫을 수 있어, 정보 처리 속도가 다른 사용자들을 지원합니다.</Section.ListItem>
    <Section.ListItem>사용자 액션이 필요한 중요한 정보에는 Toast를 사용하지 마세요. 자동 닫힘 타이머로 인해 메시지를 놓칠 수 있습니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Toaster, toast } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>사용법</Section.Title>
  <Section.Text><code>@cocso-ui/react</code>에서 <code>toast</code>와 <code>Toaster</code>를 가져옵니다. 루트 레이아웃에 <code>&lt;Toaster /&gt;</code>를 추가한 후, 알림을 표시할 곳에서 <code>toast()</code>를 호출하세요.</Section.Text>
  <ComponentExample name="toast-default">
```tsx
"use client";

import { Button, Toaster, toast } from "@cocso-ui/react";

export default function ToastDefault() {
  return (
    <div className="flex flex-wrap gap-3 p-4">
      <Button onClick={() => toast("This is a toast message")} variant="primary">
        Show Toast
      </Button>
      <Button onClick={() => toast.success("Operation successful!")} variant="success">
        Success
      </Button>
      <Button onClick={() => toast.error("Something went wrong.")} variant="error">
        Error
      </Button>
      <Toaster />
    </div>
  );
}
```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Variant</Section.Title>
  <Section.Text>시맨틱 변형 메서드를 사용해 알림의 의도를 한눈에 전달하세요.</Section.Text>
  <Section.UnorderedList>
    <Section.ListItem><code>toast(message)</code> — 기본 중립 알림입니다.</Section.ListItem>
    <Section.ListItem><code>toast.success(message)</code> — 완료된 액션을 확인합니다.</Section.ListItem>
    <Section.ListItem><code>toast.error(message)</code> — 실패 또는 오류를 알립니다.</Section.ListItem>
    <Section.ListItem><code>toast.info(message)</code> — 정보성 컨텍스트를 제공합니다.</Section.ListItem>
    <Section.ListItem><code>toast.warning(message)</code> — 잠재적으로 원하지 않는 결과에 대해 경고합니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>
