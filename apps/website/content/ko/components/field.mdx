---
title: Field
description: 레이블, 설명, 에러 메시지를 폼 컨트롤과 연결하는 폼 필드 래퍼입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>Field는 접근 가능한 레이블과 선택적 도움말 텍스트로 폼 컨트롤을 감쌉니다. 추가 안내를 위한 description과 유효성 검사 피드백을 위한 error 메시지를 제공합니다. description과 error 중 하나만 표시되며, error가 우선합니다. Input과 Select는 <code>label</code> prop이 제공되면 이미 내부적으로 Field를 통합합니다. 커스텀 폼 컨트롤을 감싸거나 독립적인 레이블/에러/설명 래퍼가 필요할 때 Field를 직접 사용하세요.</Section.Text>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>스크린 리더 접근성을 위해 <code>htmlFor</code>/<code>id</code>를 통해 <code>&lt;label&gt;</code> 요소를 자식 컨트롤과 연결합니다.</Section.ListItem>
    <Section.ListItem><code>error</code>와 <code>description</code> 텍스트는 <code>aria-describedby</code>를 통해 자식 컨트롤에 연결됩니다.</Section.ListItem>
    <Section.ListItem><code>error</code>와 <code>description</code> 중 하나만 렌더링됩니다. 둘 다 제공된 경우 <code>error</code>가 우선합니다.</Section.ListItem>
    <Section.ListItem><code>required</code>가 <code>false</code>이면 레이블 옆에 "(선택)" 표시가 나타납니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Field } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <Section.Text>자식으로 폼 컨트롤을 전달하세요. <code>label</code> prop은 필수입니다. 도움말 텍스트에는 <code>description</code>을, 유효성 검사 에러에는 <code>error</code>를 사용하세요.</Section.Text>
  <ComponentExample name="field-default">
    ```tsx
    import { Field, Input } from "@cocso-ui/react";

    export default function FieldDefault() {
      return (
        <div className="flex w-full max-w-sm flex-col gap-4 p-4">
          <Field label="Email" description="We'll never share your email.">
            <Input placeholder="Enter your email" />
          </Field>
          <Field label="Password" required error="Password must be at least 8 characters.">
            <Input placeholder="Enter password" type="password" />
          </Field>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
