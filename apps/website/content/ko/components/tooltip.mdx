---
title: Tooltip
description: 요소에 호버하거나 포커스할 때 보조 정보를 제공하는 플로팅 레이블입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>
    Tooltip은 사용자가 트리거 요소에 호버하거나 포커스할 때 짧은 텍스트 레이블을 렌더링하는 복합 컴포넌트입니다. 여러 툴팁을 공유된 <code>Tooltip.Provider</code>로 감싸면 지연 및 열기 동작을 일괄 조정할 수 있습니다.
  </Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>복합 API로 구성됩니다. <code>Tooltip.Provider</code>, <code>Tooltip</code>, <code>Tooltip.Trigger</code>, <code>Tooltip.Content</code>, <code>Tooltip.Arrow</code></Section.ListItem>
    <Section.ListItem>툴팁 그룹을 <code>Tooltip.Provider</code>로 감싸면 호버 지연 설정을 공유할 수 있습니다.</Section.ListItem>
    <Section.ListItem><code>Tooltip.Content</code>에서 <code>side</code> 및 <code>align</code> 위치 지정 props를 지원합니다.</Section.ListItem>
    <Section.ListItem><code>Tooltip.Arrow</code>는 트리거를 향하는 화살표 캐럿을 렌더링합니다.</Section.ListItem>
  </Section.UnorderedList>
  <Section.Text className="font-semibold">가이드라인</Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>최대 1~2문장의 짧은 보조 텍스트 레이블에 Tooltip을 사용합니다.</Section.ListItem>
    <Section.ListItem>툴팁 콘텐츠는 UI를 이해하는 데 필수적이지 않아야 하며 트리거 레이블을 보완하는 용도로 사용합니다.</Section.ListItem>
    <Section.ListItem>링크, 버튼, 폼 같은 인터랙티브 콘텐츠에는 Popover를 사용합니다.</Section.ListItem>
    <Section.ListItem>인근의 여러 툴팁은 반드시 공유된 <code>Tooltip.Provider</code>로 감싸 호버 지연을 일괄 조정합니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>툴팁 콘텐츠는 <code>role="tooltip"</code>으로 노출되며, 트리거는 <code>aria-describedby</code>로 연결됩니다.</Section.ListItem>
    <Section.ListItem>hover와 focus 모두에서 표시되어 키보드 사용자도 툴팁 콘텐츠에 접근할 수 있습니다.</Section.ListItem>
    <Section.ListItem>트리거의 hover 또는 focus가 해제되거나 <code>Escape</code>를 누르면 사라집니다.</Section.ListItem>
    <Section.ListItem>툴팁 콘텐츠는 보조적이어야 합니다. 중요한 정보를 툴팁에만 배치하지 마세요. 일부 사용자는 발견하지 못할 수 있습니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Tooltip } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <Section.Text>트리거 버튼에 호버할 때 나타나는 툴팁입니다.</Section.Text>
  <ComponentExample name="tooltip-default">
```tsx
"use client";

import { Button, Tooltip } from "@cocso-ui/react";

export default function TooltipDefault() {
  return (
    <div className="p-4">
      <Tooltip.Provider>
        <Tooltip>
          <Tooltip.Trigger render={<Button variant="secondary" />}>
            Hover me
          </Tooltip.Trigger>
          <Tooltip.Content>
            This is a tooltip
            <Tooltip.Arrow />
          </Tooltip.Content>
        </Tooltip>
      </Tooltip.Provider>
    </div>
  );
}
```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>위치</Section.Title>
  <Section.Text><code>Tooltip.Content</code>의 <code>side</code> prop을 사용해 툴팁이 트리거의 어느 방향에 표시될지 제어합니다.</Section.Text>
  <ComponentExample name="tooltip-positions">
```tsx
"use client";

import { Button, Tooltip } from "@cocso-ui/react";

export default function TooltipPositions() {
  return (
    <div className="flex flex-wrap gap-3 p-4">
      <Tooltip.Provider>
        {(["top", "bottom", "left", "right"] as const).map((side) => (
          <Tooltip key={side}>
            <Tooltip.Trigger render={<Button variant="outline" />}>{side}</Tooltip.Trigger>
            <Tooltip.Content side={side}>
              Tooltip on {side}
              <Tooltip.Arrow />
            </Tooltip.Content>
          </Tooltip>
        ))}
      </Tooltip.Provider>
    </div>
  );
}
```
  </ComponentExample>
</Section>
