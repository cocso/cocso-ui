---
title: Day Picker
description: 단일 날짜 선택을 위한 달력 팝오버를 여는 날짜 선택기입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>DayPicker는 인라인 달력이 포함된 Dropdown을 여는 트리거 요소를 렌더링합니다. 제어 컴포넌트로, <code>value</code>와 <code>onValueChange</code>를 전달하여 선택된 날짜를 관리합니다. 달력은 기본적으로 한국어로 현지화됩니다.</Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>특정 달력 날짜 선택에 DayPicker를 사용합니다.</Section.ListItem>
    <Section.ListItem><code>minDate</code>와 <code>maxDate</code>로 선택 가능한 범위를 제한할 수 있습니다.</Section.ListItem>
    <Section.ListItem>월 단위 선택이 필요한 경우 MonthPicker를 사용합니다.</Section.ListItem>
    <Section.ListItem>폼 레이아웃과의 시각적 통합을 위해 커스텀 <code>trigger</code> 요소를 제공합니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>달력 그리드에서 화살표 키로 날짜 간 탐색할 수 있습니다.</Section.ListItem>
    <Section.ListItem><code>Escape</code>로 드롭다운을 닫고 트리거로 포커스를 반환합니다.</Section.ListItem>
    <Section.ListItem><code>minDate</code>–<code>maxDate</code> 범위 밖의 날짜는 <code>aria-disabled</code>로 표시됩니다.</Section.ListItem>
    <Section.ListItem>월 탐색 버튼으로 월을 앞뒤로 이동할 수 있습니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { DayPicker } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <Section.Text><code>value</code>와 <code>onValueChange</code> props로 선택된 날짜를 제어하세요. 커스텀 트리거가 제공되지 않으면 기본 버튼으로 렌더링됩니다.</Section.Text>
  <ComponentExample name="day-picker-default">
    ```tsx
    "use client";

    import { Button, DayPicker } from "@cocso-ui/react";
    import { CalendarMonthIcon } from "@cocso-ui/react-icons";
    import { useState } from "react";

    export default function DayPickerDefault() {
      const [value, setValue] = useState<Date | null>(null);

      return (
        <div className="p-4">
          <DayPicker
            onValueChange={setValue}
            trigger={
              <Button prefix={<CalendarMonthIcon size={16} />} variant="outline">
                {value ? value.toLocaleDateString("ko-KR") : "Select date"}
              </Button>
            }
            value={value ?? undefined}
          />
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
