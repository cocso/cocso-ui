title: Tab
description: 관련 콘텐츠 패널 간 전환을 위한 탭 인터페이스입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>Tab은 WAI-ARIA 패턴을 따르는 키보드 접근 가능한 탭 인터페이스를 제공하는 복합 컴포넌트입니다. 콘텐츠 패널은 공유된 `value` prop을 통해 해당 트리거와 연결됩니다.</Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>같은 페이지 컨텍스트 내에서 관련 콘텐츠 뷰 간 전환(예: 다른 설정 카테고리나 데이터 뷰)에 Tab을 사용합니다.</Section.ListItem>
    <Section.ListItem>단계별 워크플로에는 stepper 패턴을 사용합니다.</Section.ListItem>
    <Section.ListItem>탭 안에 탭을 중첩하지 않습니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem><code>role="tablist"</code>, <code>role="tab"</code>, <code>role="tabpanel"</code>을 사용하는 WAI-ARIA Tabs 패턴을 따릅니다.</Section.ListItem>
    <Section.ListItem><code>Arrow Left</code> / <code>Arrow Right</code>로 탭 트리거 간 탐색합니다.</Section.ListItem>
    <Section.ListItem><code>Home</code>으로 첫 번째 탭으로 포커스를 이동합니다. <code>End</code>로 마지막 탭으로 이동합니다.</Section.ListItem>
    <Section.ListItem>탭 패널은 <code>aria-labelledby</code>를 통해 트리거와 연결되어 스크린 리더에 컨텍스트를 제공합니다.</Section.ListItem>
    <Section.ListItem>활성화된 탭 패널만 접근성 트리에 렌더링되며, 비활성 패널은 숨겨집니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Tab } from "@cocso-ui/react";
  ```
</Section>
