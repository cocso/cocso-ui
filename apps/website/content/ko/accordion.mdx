---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>Accordion은 WAI-ARIA disclosure 위젯 패턴을 따르는 복합 컴포넌트입니다. 기본적으로 키보드 탐색을 지원합니다. 각 아이템은 독립적으로 펼치거나 접을 수 있습니다. FAQ, 설정 패널, 공간이 제한된 상세 섹션 등 접을 수 있는 콘텐츠 영역에 Accordion을 사용하세요. 헤더는 간결하고 명확하게 작성하세요. 숨겨진 콘텐츠의 주요 레이블 역할을 합니다.</Section.Text>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>WAI-ARIA Accordion (disclosure widget) 패턴을 따릅니다.</Section.ListItem>
    <Section.ListItem><code>Enter</code> 또는 <code>Space</code>로 포커스된 아이템을 열거나 닫을 수 있습니다.</Section.ListItem>
    <Section.ListItem><code>Arrow Down</code>으로 다음 헤더로 포커스를 이동합니다. <code>Arrow Up</code>으로 이전 헤더로 이동합니다.</Section.ListItem>
    <Section.ListItem><code>Home</code>으로 첫 번째 헤더로 포커스를 이동합니다. <code>End</code>로 마지막 헤더로 이동합니다.</Section.ListItem>
    <Section.ListItem>각 트리거에는 콘텐츠 패널과 연결하는 <code>aria-expanded</code>와 <code>aria-controls</code>가 있습니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Accordion } from "@cocso-ui/react";
  ```
</Section>
