---
title: Month Picker
description: 월과 연도 선택을 위한 달력 팝오버를 여는 날짜 선택기입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>MonthPicker는 월-연도 그리드가 포함된 Dropdown을 여는 트리거 요소를 렌더링합니다. 제어 컴포넌트로, <code>value</code>와 <code>onValueChange</code>를 전달하여 선택된 월을 관리합니다. 탐색 화살표는 월이 아닌 연도 단위로 이동합니다. 달력은 기본적으로 한국어로 현지화됩니다. 특정 날짜가 아닌 월 단위 세분성이 필요한 경우(예: 청구 기간, 리포트 범위, 구독 주기)에 MonthPicker를 사용하세요. 일 단위 선택이 필요한 경우 DayPicker를 사용하세요.</Section.Text>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>월 그리드에서 키보드로 월 간 탐색할 수 있습니다.</Section.ListItem>
    <Section.ListItem><code>Escape</code>로 드롭다운을 닫고 트리거로 포커스를 반환합니다.</Section.ListItem>
    <Section.ListItem>연도 탐색 화살표로 연도를 앞뒤로 이동할 수 있습니다.</Section.ListItem>
    <Section.ListItem><code>minDate</code>–<code>maxDate</code> 범위 밖의 월은 시각적으로 흐리게 표시되며 선택할 수 없습니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { MonthPicker } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <Section.Text><code>value</code>와 <code>onValueChange</code> props로 선택된 월을 제어하세요. 반환되는 Date는 선택된 월의 첫째 날을 나타냅니다.</Section.Text>
  <ComponentExample name="month-picker-default">
    ```tsx
    "use client";

    import { MonthPicker } from "@cocso-ui/react";
    import { useState } from "react";

    export default function MonthPickerDefault() {
      const [value, setValue] = useState<Date | null>(null);

      return (
        <div className="p-4">
          <MonthPicker onValueChange={setValue} value={value ?? undefined} />
          {value && <p className="mt-2 text-sm text-neutral-500">Selected: {value.toLocaleDateString("en-US", { year: "numeric", month: "long" })}</p>}
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
