---
title: Switch
description: 이진 켜기/끄기 설정을 위한 토글 컨트롤입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>
    Switch 컴포넌트는 설정을 활성화하거나 비활성화하는 시각적 토글을 제공합니다. 헤드리스 접근성 기반으로 구축되었으며, 다양한 크기, 시맨틱 색상 변형, 그리고 레이블 위치 설정을 지원합니다.
  </Section.Text>
  <Section.Text>
    Switch는 즉각적인 토글을 나타냅니다. 폼 제출 없이 즉시 적용되는 설정에 사용하세요 (예: "알림 활성화", "다크 모드"). 저장 액션이 필요한 변경에는 Checkbox를 사용하세요.
  </Section.Text>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>보조 기술을 위해 <code>role="switch"</code>와 <code>aria-checked</code>를 지원합니다.</Section.ListItem>
    <Section.ListItem><code>Space</code>로 스위치를 토글합니다. <code>Tab</code>으로 다음 포커스 가능한 요소로 이동합니다.</Section.ListItem>
    <Section.ListItem><code>label</code>이 제공되면 <code>id</code> 속성을 통해 스위치와 자동으로 연결됩니다.</Section.ListItem>
    <Section.ListItem><code>disabled</code> 상태에서는 스위치가 시각적으로 흐리게 표시되고 키보드 탐색에서 제외됩니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Switch } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <Section.Text>
    로컬 상태로 제어되는 기본 스위치입니다.
  </Section.Text>
  <ComponentExample name="switch-default">
    ```tsx
    "use client";

    import { Switch } from "@cocso-ui/react";
    import { useState } from "react";

    export default function SwitchDefault() {
      const [checked, setChecked] = useState(false);

      return (
        <div className="flex flex-col gap-4 p-4">
          <Switch checked={checked} label="Enable notifications" onCheckedChange={setChecked} />
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>크기</Section.Title>
  <Section.Text>
    세 가지 크기를 사용할 수 있습니다. 주변 레이아웃 밀도에 따라 선택하세요.
  </Section.Text>
  <ComponentExample name="switch-sizes">
    ```tsx
    import { Switch } from "@cocso-ui/react";

    export default function SwitchSizes() {
      return (
        <div className="flex flex-col gap-4 p-4">
          <Switch defaultChecked label="Large" size="large" />
          <Switch defaultChecked label="Medium" size="medium" />
          <Switch defaultChecked label="Small" size="small" />
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Variant</Section.Title>
  <Section.Text>
    색상 변형은 토글의 시맨틱 의미를 전달합니다. 일반 설정에는 <code>primary</code>를, 특정 결과가 있는 액션에는 상태 변형을 사용하세요.
  </Section.Text>
  <ComponentExample name="switch-variants">
    ```tsx
    import { Switch } from "@cocso-ui/react";

    export default function SwitchVariants() {
      return (
        <div className="flex flex-col gap-4 p-4">
          <Switch defaultChecked label="Primary" variant="primary" />
          <Switch defaultChecked label="Success" variant="success" />
          <Switch defaultChecked label="Error" variant="error" />
          <Switch defaultChecked label="Warning" variant="warning" />
          <Switch defaultChecked label="Info" variant="info" />
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>레이블 위치</Section.Title>
  <Section.Text>
    <code>position</code> prop을 사용해 레이블이 스위치의 왼쪽 또는 오른쪽에 표시될지 제어합니다. 기본값은 <code>right</code>입니다.
  </Section.Text>
</Section>

<Section>
  <Section.Title>비활성화</Section.Title>
  <Section.Text>
    <code>disabled</code>를 설정하면 사용자가 스위치를 토글하지 못하도록 합니다.
  </Section.Text>
</Section>
