---
title: Input
description: 사용자 입력을 수집하기 위한 텍스트 필드입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>
    Input 컴포넌트는 한 줄 텍스트 필드입니다. 레이블, 설명, 에러 메시지, 다양한 크기를 지원합니다. 폼, 검색 필드, 자유 형식의 텍스트 입력이 필요한 모든 시나리오에 사용하세요. 네이티브 HTML input 래퍼로서 제어형(<code>value</code> + <code>onChange</code>)과 비제어형(<code>defaultValue</code>) 사용을 모두 지원합니다.
  </Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>접근성을 위해 항상 <code>label</code>을 제공합니다. Field 래퍼를 통해 가시적인 <code>&lt;label&gt;</code> 요소를 입력 필드와 연결합니다.</Section.ListItem>
    <Section.ListItem>형식 요구사항과 같은 지속적인 도움말 텍스트에는 <code>description</code>을 사용합니다.</Section.ListItem>
    <Section.ListItem><code>error</code>는 상호작용 이후 유효성 검사 피드백에만 사용합니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem><code>label</code>이 제공되면 입력 필드는 자동으로 <code>Field</code> 컴포넌트로 감싸지며, <code>&lt;label&gt;</code> 요소가 입력 필드와 연결됩니다.</Section.ListItem>
    <Section.ListItem><code>error</code>와 <code>description</code> 메시지는 <code>aria-describedby</code>를 통해 입력 필드에 연결되어, 스크린 리더가 이를 안내합니다.</Section.ListItem>
    <Section.ListItem><code>id</code>가 제공되지 않으면 자동 생성된 <code>id</code>가 사용되어 레이블-입력 필드 연결이 유지됩니다.</Section.ListItem>
    <Section.ListItem><code>disabled</code> 상태에서는 입력 필드에 포커스하거나 편집할 수 없으며, 폼 제출에서 제외됩니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Input } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <Section.Text>
    레이블과 플레이스홀더가 있는 기본 입력 필드입니다.
  </Section.Text>
  <ComponentExample name="input-default">
    ```tsx
    import { Input } from "@cocso-ui/react";

    export default function InputDefault() {
      return (
        <div className="flex w-full max-w-sm flex-col gap-4 p-4">
          <Input label="Email" placeholder="Enter your email" />
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>크기</Section.Title>
  <Section.Text>
    주변 콘텐츠 밀도에 맞게 네 가지 크기를 사용할 수 있습니다.
  </Section.Text>
  <ComponentExample name="input-sizes">
    ```tsx
    import { Input } from "@cocso-ui/react";

    export default function InputSizes() {
      return (
        <div className="flex w-full max-w-sm flex-col gap-4 p-4">
          <Input label="Large" placeholder="Large input" size="large" />
          <Input label="Medium" placeholder="Medium input" size="medium" />
          <Input label="Small" placeholder="Small input" size="small" />
          <Input label="X-Small" placeholder="X-Small input" size="x-small" />
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>상태</Section.Title>
  <Section.Text>
    도움말 텍스트에는 <code>description</code>, 유효성 검사 메시지에는 <code>error</code>, 상호작용 방지에는 <code>disabled</code>를 사용합니다.
  </Section.Text>
  <ComponentExample name="input-states">
    ```tsx
    import { Input } from "@cocso-ui/react";

    export default function InputStates() {
      return (
        <div className="flex w-full max-w-sm flex-col gap-4 p-4">
          <Input label="With description" description="We'll never share your email." placeholder="Email" />
          <Input error="This field is required." label="With error" placeholder="Required field" />
          <Input disabled label="Disabled" placeholder="Cannot edit" />
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>

  </Section.Text>
</Section>
