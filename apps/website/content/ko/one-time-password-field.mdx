---
title: One-Time Password Field
description: 일회용 비밀번호 및 인증 코드 입력을 위한 분할 입력 필드입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>OneTimePasswordField는 일회용 비밀번호 또는 인증 코드를 입력하기 위한 개별 입력 슬롯 행을 렌더링합니다. input-otp를 기반으로 만들어졌으며, 붙여넣기, 키보드 탐색, 접근성을 기본으로 지원합니다. onValueChange를 사용하여 현재 전체 값을 문자열로 받으세요. SMS, 이메일, 인증 앱의 인증 코드에 사용하세요. 완전한 인증 UX를 위해 타이머나 "코드 재전송" 액션과 함께 사용하세요.</Section.Text>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>코드 입력을 위한 적절한 ARIA 속성을 갖춘 <code>input-otp</code> 기반으로 구축되었습니다.</Section.ListItem>
    <Section.ListItem>마운트 시 첫 번째 빈 슬롯에 자동으로 포커스하여 즉시 입력할 수 있습니다.</Section.ListItem>
    <Section.ListItem>붙여넣기를 지원합니다. 사용자가 전체 코드를 한 번에 붙여넣으면 슬롯에 자동으로 분배됩니다.</Section.ListItem>
    <Section.ListItem><code>Tab</code>으로 OTP 필드를 단일 단위로 진입하거나 빠져나옵니다.</Section.ListItem>
    <Section.ListItem>화살표 키로 수정을 위해 개별 슬롯 간에 이동합니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { OneTimePasswordField } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <ComponentExample name="otp-default">
    ```tsx
    "use client";

    import { OneTimePasswordField } from "@cocso-ui/react";
    import { useState } from "react";

    export default function OTPDefault() {
      const [value, setValue] = useState("");

      return (
        <div className="p-4">
          <OneTimePasswordField onValueChange={setValue} />
          {value && <p className="mt-2 text-sm text-neutral-500">Value: {value}</p>}
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
