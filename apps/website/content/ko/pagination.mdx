---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>Pagination은 이전/다음 컨트롤과 함께 페이지 번호 버튼 행을 렌더링합니다. 제어 컴포넌트로, 부모가 `page`와 `onChange`를 통해 현재 페이지를 관리합니다. 표시되는 페이지 버튼 수는 `maxVisible`로 설정할 수 있습니다.</Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>대규모 데이터셋의 페이지 이동에 사용합니다.</Section.ListItem>
    <Section.ListItem>기본 `maxVisible` 값인 5는 대부분의 레이아웃에 적합합니다. 좁은 컨테이너나 모바일 뷰에서는 줄입니다.</Section.ListItem>
    <Section.ListItem>완전한 페이지네이션 인터페이스를 위해 데이터 테이블이나 리스트 컴포넌트와 함께 사용합니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>랜드마크 네비게이션을 위해 <code>&lt;nav&gt;</code> 요소 안에 렌더링됩니다.</Section.ListItem>
    <Section.ListItem>현재 페이지 버튼에는 <code>aria-current="page"</code>가 있어 스크린 리더에 활성 페이지를 전달합니다.</Section.ListItem>
    <Section.ListItem>이전 및 다음 버튼은 첫 번째 및 마지막 페이지에서 각각 자동으로 비활성화됩니다.</Section.ListItem>
    <Section.ListItem>각 페이지 버튼은 <code>Tab</code>으로 키보드 포커스가 가능하며 <code>Enter</code> 또는 <code>Space</code>로 활성화할 수 있습니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Pagination } from "@cocso-ui/react";
  ```
</Section>
