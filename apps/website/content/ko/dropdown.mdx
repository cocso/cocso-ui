---
title: Dropdown
description: 트리거 활성화 시 나타나는 컨텍스트 메뉴로, Base UI Menu 프리미티브 기반입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>
    Dropdown은 트리거 요소에 고정된 플로팅 액션 목록을 렌더링하는 복합 컴포넌트입니다. 편집, 복제, 삭제 같은 컨텍스트 액션이나 특정 항목에 대한 명령 목록에 사용하세요.
  </Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>복합 API로 구성됩니다. <code>Dropdown</code>, <code>Dropdown.Trigger</code>, <code>Dropdown.Content</code>, <code>Dropdown.Item</code></Section.ListItem>
    <Section.ListItem>화살표 키로 키보드 탐색이 가능하며, <code>Escape</code> 키로 메뉴를 닫습니다.</Section.ListItem>
    <Section.ListItem>아이템은 선행 아이콘을 위한 <code>prefix</code> 슬롯을 지원합니다.</Section.ListItem>
  </Section.UnorderedList>
  <Section.UnorderedList>
    <Section.ListItem>편집, 복제, 삭제 등 특정 항목과 연결된 컨텍스트 액션에 Dropdown을 사용합니다.</Section.ListItem>
    <Section.ListItem>탐색 메뉴나 콘텐츠 선택에는 Select 또는 Tab을 사용합니다.</Section.ListItem>
    <Section.ListItem>최적의 사용성을 위해 메뉴 항목은 3~7개로 유지합니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>접근성</Section.Title>
  <Section.UnorderedList>
    <Section.ListItem>올바른 ARIA 시맨틱을 위해 <code>role="menu"</code>와 <code>role="menuitem"</code>을 사용합니다.</Section.ListItem>
    <Section.ListItem><code>Arrow Down</code> / <code>Arrow Up</code>으로 항목 간 이동합니다.</Section.ListItem>
    <Section.ListItem><code>Enter</code> 또는 <code>Space</code>로 포커스된 항목을 활성화합니다.</Section.ListItem>
    <Section.ListItem><code>Escape</code>로 메뉴를 닫고 트리거로 포커스가 돌아갑니다.</Section.ListItem>
    <Section.ListItem>타입 어헤드를 지원합니다. 문자를 입력하면 해당 문자로 시작하는 첫 번째 항목으로 포커스가 이동합니다.</Section.ListItem>
    <Section.ListItem>트리거 요소에 <code>aria-expanded</code>와 <code>aria-haspopup="menu"</code> 속성이 적용됩니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>Import</Section.Title>
  ```tsx
  import { Dropdown } from "@cocso-ui/react";
  ```
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <Section.Text>아이콘 버튼으로 트리거되며 레이블이 있는 액션 아이템을 포함한 드롭다운 메뉴입니다.</Section.Text>
  <ComponentExample name="dropdown-default">
```tsx
"use client";

import { Button, Dropdown } from "@cocso-ui/react";
import { ContentCopyIcon, DeleteIcon, MoreHorizIcon, PencilIcon } from "@cocso-ui/react-icons";

export default function DropdownDefault() {
  return (
    <div className="p-4">
      <Dropdown>
        <Dropdown.Trigger render={<Button variant="secondary" svgOnly />}>
          <MoreHorizIcon size={18} />
        </Dropdown.Trigger>
        <Dropdown.Content>
          <Dropdown.Item prefix={<PencilIcon size={16} />}>Edit</Dropdown.Item>
          <Dropdown.Item prefix={<ContentCopyIcon size={16} />}>Duplicate</Dropdown.Item>
          <Dropdown.Item prefix={<DeleteIcon size={16} />}>Delete</Dropdown.Item>
        </Dropdown.Content>
      </Dropdown>
    </div>
  );
}
```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Prefix Icon</Section.Title>
  <Section.Text><code>Dropdown.Item</code>의 <code>prefix</code> prop에 아이콘 노드를 전달하면 레이블 앞에 선행 아이콘을 렌더링합니다.</Section.Text>
</Section>
