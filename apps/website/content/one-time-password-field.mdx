---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>OneTimePasswordField는 일회용 비밀번호 또는 인증 코드를 입력하기 위한 개별 입력 슬롯 행을 렌더링합니다. input-otp를 기반으로 만들어졌으며, 붙여넣기, 키보드 탐색, 접근성을 기본으로 지원합니다. onValueChange를 사용하여 현재 전체 값을 문자열로 받으세요.</Section.Text>
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <ComponentExample name="otp-default">
    ```tsx
    "use client";

    import { OneTimePasswordField } from "@cocso-ui/react";
    import { useState } from "react";

    export default function OTPDefault() {
      const [value, setValue] = useState("");

      return (
        <div className="p-4">
          <OneTimePasswordField onValueChange={setValue} />
          {value && <p className="mt-2 text-sm text-neutral-500">Value: {value}</p>}
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
