---
title: Typography Component
description: custom, body, heading 모드를 지원하는 유연한 텍스트 렌더링 컴포넌트입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>Typography 컴포넌트는 type prop으로 제어되는 세 가지 모드로 동작합니다. custom은 임의의 숫자 폰트 크기, body는 이름 있는 크기 단계를 가진 시맨틱 본문 텍스트, heading은 반응형 크기 단계를 가진 시맨틱 제목입니다. 모든 모드는 weight, lineHeight, 기본 요소를 교체하는 render prop을 지원합니다.</Section.Text>
</Section>

<Section>
  <Section.Title>Heading</Section.Title>
  <Section.Text>heading 모드는 기본적으로 h2 요소를 렌더링합니다. x-large와 large 크기는 반응형으로, 태블릿 브레이크포인트에서 크기가 증가합니다.</Section.Text>
  <ComponentExample name="typography-heading">
    ```tsx
    import { Typography } from "@cocso-ui/react";

    export default function TypographyHeading() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Typography type="heading" size="x-large">X-Large Heading</Typography>
          <Typography type="heading" size="large">Large Heading</Typography>
          <Typography type="heading" size="medium">Medium Heading</Typography>
          <Typography type="heading" size="small">Small Heading</Typography>
          <Typography type="heading" size="x-small">X-Small Heading</Typography>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Body</Section.Title>
  <Section.Text>body 모드는 p 요소를 렌더링합니다. 모든 표준 산문 및 UI 텍스트에 사용하세요.</Section.Text>
  <ComponentExample name="typography-body">
    ```tsx
    import { Typography } from "@cocso-ui/react";

    export default function TypographyBody() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Typography type="body" size="large">Large body text for prominent content.</Typography>
          <Typography type="body" size="medium">Medium body text for standard content.</Typography>
          <Typography type="body" size="small">Small body text for secondary content.</Typography>
          <Typography type="body" size="x-small">X-Small body text for captions.</Typography>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Custom</Section.Title>
  <Section.Text>custom 모드(기본값)는 디자인 토큰 스케일의 임의 숫자 폰트 크기, 또는 base, tablet, desktop 키를 가진 반응형 객체를 받습니다.</Section.Text>
  <ComponentExample name="typography-custom">
    ```tsx
    import { Typography } from "@cocso-ui/react";

    export default function TypographyCustom() {
      return (
        <div className="flex flex-col gap-3 p-4">
          <Typography size={24} weight="bold">Custom size 24, bold</Typography>
          <Typography size={16} weight="medium">Custom size 16, medium</Typography>
          <Typography size={12} weight="normal" lineHeight="relaxed">Custom size 12, relaxed line height</Typography>
        </div>
      );
    }
    ```
  </ComponentExample>
</Section>
