---
title: Dropdown
description: 트리거 활성화 시 나타나는 컨텍스트 메뉴로, Base UI Menu 프리미티브 기반입니다.
---

<Section>
  <Section.Title>개요</Section.Title>
  <Section.Text>
    Dropdown은 Base UI의 Menu 프리미티브 기반으로 구성된 복합 컴포넌트입니다. 트리거 요소에 고정된 플로팅 액션 목록을 렌더링합니다. 편집, 복제, 삭제 같은 컨텍스트 액션이나 특정 항목에 대한 명령 목록에 사용하세요.
  </Section.Text>
  <Section.UnorderedList>
    <Section.ListItem>복합 API로 구성됩니다. <code>Dropdown</code>, <code>Dropdown.Trigger</code>, <code>Dropdown.Content</code>, <code>Dropdown.Item</code></Section.ListItem>
    <Section.ListItem>화살표 키로 키보드 탐색이 가능하며, <code>Escape</code> 키로 메뉴를 닫습니다.</Section.ListItem>
    <Section.ListItem>아이템은 선행 아이콘을 위한 <code>prefix</code> 슬롯을 지원합니다.</Section.ListItem>
  </Section.UnorderedList>
</Section>

<Section>
  <Section.Title>기본</Section.Title>
  <Section.Text>아이콘 버튼으로 트리거되며 레이블이 있는 액션 아이템을 포함한 드롭다운 메뉴입니다.</Section.Text>
  <ComponentExample name="dropdown-default">
```tsx
"use client";

import { Button, Dropdown } from "@cocso-ui/react";
import { ContentCopyIcon, DeleteIcon, MoreHorizIcon, PencilIcon } from "@cocso-ui/react-icons";

export default function DropdownDefault() {
  return (
    <div className="p-4">
      <Dropdown>
        <Dropdown.Trigger render={<Button variant="secondary" svgOnly />}>
          <MoreHorizIcon size={18} />
        </Dropdown.Trigger>
        <Dropdown.Content>
          <Dropdown.Item prefix={<PencilIcon size={16} />}>Edit</Dropdown.Item>
          <Dropdown.Item prefix={<ContentCopyIcon size={16} />}>Duplicate</Dropdown.Item>
          <Dropdown.Item prefix={<DeleteIcon size={16} />}>Delete</Dropdown.Item>
        </Dropdown.Content>
      </Dropdown>
    </div>
  );
}
```
  </ComponentExample>
</Section>

<Section>
  <Section.Title>Prefix Icon</Section.Title>
  <Section.Text><code>Dropdown.Item</code>의 <code>prefix</code> prop에 아이콘 노드를 전달하면 레이블 앞에 선행 아이콘을 렌더링합니다.</Section.Text>
</Section>
